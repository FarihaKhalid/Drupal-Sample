Drupal 8 Examples: Popup

INSTALLATION & USAGE
---------------------

1) Download and Install the module.
2) Go to extend and enable the module.
3) After enabling the module, visit the URL : '/popup'. For example : www.example.com/popup
4) Click on "Click Me" link, Popup appears.
5) Currently we have given node/1 in the link. You can give whatever node id or block id you like.

More Information
-----------------

We have used default Drupal Modal Dialog. Please find the link here : https://api.drupal.org/api/drupal/core%21lib%21Drupal%21Core%21Ajax%21OpenModalDialogCommand.php/class/OpenModalDialogCommand/8.2.x
